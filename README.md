## Script Organizador de Arquivos (Bash)

## Objetivo

Este projeto foi criado para praticar scripts Bash através da automação da organização de arquivos.

## O que o script faz

- Escaneia arquivos no diretório atual
- Identifica extensões de arquivos
- Cria pastas baseadas nas extensões
- Move arquivos para suas respectivas pastas
- Lida com arquivos sem extensão
- Gera um arquivo de log com as ações executadas
- Conta os arquivos processados

## Como usar

bash organizer.sh

## Aprendizados

- Melhor compreensão de loops e condicionais em Bash
- Aprendi a manipular strings em Bash
- Pratiquei operações de arquivo e diretório (mv, mkdir)
- Ganhei experiência na depuração de shell scripts

## Problemas Encontrados

- Problemas com escopo de variáveis e blocos condicionais
- Manipulação incorreta de extensões de arquivo
- Compreensão incorreta do comportamento de redirecionamento (>>)
- Problemas relacionados a caminhos (paths) ao escrever logs
